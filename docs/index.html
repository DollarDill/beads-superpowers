What is this?

AI coding agents have two recurring failure modes: no process discipline (they skip tests, rush to code, and claim work is done without verification) and no persistent memory (todo lists vanish when a session ends, the next session starts blind).

beads-superpowers is a Claude Code plugin that solves both at once by merging two upstream systems:

Superpowers — Process Discipline

20 mandatory skills enforcing TDD, brainstorming, systematic debugging, two-stage code review, and verification. Skills are not suggestions — they use bright-line rules and anti-rationalization tables.

Beads — Persistent Memory

A Dolt-backed issue tracker that survives across sessions. Every task is a bead tracked in a persistent ledger. The agent doesn't just follow the right process — it leaves a permanent record of having done so.

20 Skills

Each skill is a composable process document that Claude Code loads on demand. The bootstrap skill (using-superpowers) routes to the right skill automatically based on the task.

Install in 60 seconds

Option A: curl (one command, recommended)

curl -fsSL https://raw.githubusercontent.com/DollarDill/beads-superpowers/main/install.sh | bash

Option B: Claude Code Marketplace

claude plugin marketplace add DollarDill/beads-superpowers
claude plugin install beads-superpowers@beads-superpowers-marketplace

Option C: npx (via Vercel Skills CLI)

npx skills add DollarDill/beads-superpowers --all -y -g

Then in any project:

cd your-project
bd init
